# heroStack 포트폴리오 웹사이트 프로젝트

이 파일은 Gemini CLI가 다음 세션에서도 기억하고 따라야 할 핵심 행동 지침 및 기획 내용입니다.

## 0. 행동 규칙
- **언어** 항상 질문과 설명은 "한국어"를 사용해서 할것.

## 1. 프로젝트 개요
- **목적:** 3D 설계 디자이너(최은진 님)의 포트폴리오 웹사이트 구축
- **디자인 컨셉:** 모던 미니멀리즘 (Modern Minimal) - 화이트 톤, 부드러운 애니메이션, 높은 가독성
- **기술 스택:** React (Vite 기반) + TypeScript + Tailwind CSS + Framer Motion
- **저장소:** https://github.com/eunjinchoi95/eunjinstack.git

## 2. 작업 내역 및 다음 단계 (Phase 1)
- PDF 포트폴리오 분석을 통해 3D 디자이너 경력(한맥아이티, 코스펙에이비, 국보디자인) 및 스킬셋을 `portfolioData.ts`에 동기화 완료했습니다.
- **다음 작업:** `App.tsx`, `About.tsx`, `Experience.tsx` 등 UI 컴포넌트의 레이블과 하이라이트 로직을 3D 디자이너 컨셉에 맞춰 수정해야 합니다.
